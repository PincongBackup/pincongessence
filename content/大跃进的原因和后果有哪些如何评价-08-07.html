大跃进的原因和后果有哪些？如何评价？

月轨

“大跃进”的成因十分的错综复杂、言人人殊,我们需从“外部因素”和“内部因素”两个方面进行检视、梳理、列举和考究,至于说这些因素孰主孰次、孰轻孰重、孰是孰非,还望各位朋友自斟自酌、独立思悟。

一.外部因素 : 1946年3月5日,英国前首相温斯顿.丘吉尔(Winston Leonard Spencer Churchill 1874 - 1965年)在威斯敏斯特学院发表了名为《和平砥柱》的演讲,明言“不久刚被盟军胜利所照亮的大地已经笼罩上了阴影。没有人知道苏俄和它的共产主义国际组织打算在最近的将来干些什么,以及它们领土与意识形态扩张倾向的止境在哪里,如果还有止境的话。①”、以及“从波罗的海的什切青到亚得里亚海边的的里雅斯特，一幅横贯欧洲大陆的铁幕已经降落下来。②”,并呼吁美国、英国、加拿大和“英语世界”的其他成员凝聚共识、构建同盟,一同应对和抵御“阴影”的贪心不足蛇吞象的扩张。1947年3月12日,美国总统杜鲁门(Harry S. Truman 1884 - 1972年)致国会的咨文中,直言“我相信,美国的政策必须是支持各自由民族,他们抵抗着企图征服他们的掌握武装的少数人或外来的压力。我相信,我们必须帮助自由民族通过他们自己的方式来安排自己的命运。我相信,我们的帮助主要是通过经济和财政的支持,这对于经济安定和有秩序的政治进程来说,是必要的。③”1947年5月22日,杜鲁门签署了《援助希(指希腊)、土(指土耳其)法案》,从此,美、苏冷战正式开始。1949年10月1日,中华人民共和国(以下简称PRC)建立于美、苏冷战的前期,一方面毛泽东与斯大林(Joseph Vissarionovich Stalin 1878 - 1953年)素来不睦、中苏之间的龃龉和分歧越来越大,另一方面中国和美国之间旧恨添新仇,1950年2月1日美国国会通过特别法案,向中华民国政府提供1.08亿美元的经济援助,且美国对中华民国政府的经济和军事援助愈来愈多、愈来愈积极,中共对此恨之入骨。可以说,1949至1958(“大跃进”运动的发动年份)年间,中共和PRC政府面临的国际形势和处境是很险恶的。基于此,我们可以得出“大跃进”运动的几点外部因素 :

1.美、苏冷战 : 如上所述,中共和PRC与美国之间的剑拔弩张是无需赘言的,可毛泽东与斯大林也互不信任和相互厌恶,即便1950年2月14日签订了中苏友好同盟互助条约及相关协议,中共和PRC政府付出了堪称“丧权辱国”的代价(比如PRC政府承认外蒙独立、中苏联合开发新疆矿产资源、共同管理长春铁路、共同使用旅顺和大连港),中、苏之间的同盟仍是形同虚设、一纸空文。直至中共和PRC政府向朝鲜半岛派遣“志愿军”、以数十万军人的生命为代价向苏联呈献了“投名状”,中、苏之间的同盟和“互助”才得到一定程度的落实。美、苏冷战造成了中共和PRC政府的左支右绌、前狼后虎的困局,因此,中共和PRC政府火急火燎的谋求自身实力的大幅提升。

2.朝韩战争和“抗美援朝” : 1950年6月25日晚,朝鲜人民军越过三八线南进,朝韩战争爆发,同年10月19日,彭德怀率领“中国人民志愿军”跨过鸭绿江并赶赴朝鲜战场,所谓“抗美援朝”由此而始。杜润生曾言“建国以来最大错误是抗美援朝”,此言不虚,人员伤亡和物资损失仅仅是“小巫”,它对PRC政府的建国方略与中共领导人的认识和判断的影响及改变才是“大巫”。

3.中、苏关系的一再恶化 : 苏联领导人的更迭也不能扭转中、苏两国关系的颓势,反而赫鲁晓夫(Nikita Sergeyevich Khrushchev 1894 - 1971年)攀上个人权力巅峰的过程,也是中、苏两国关系跌入低谷的过程。1953年,斯大林死于脑淤血,雄心勃勃的毛泽东就决心成为国际共产主义的意识形态领袖,更积极的对社会主义集团输出“毛泽东道路(毛泽东道路的两个特点:1.由工人阶级和共产党领导的全民族反帝联合战线;2.由共产党领导的人民军队,开展持久的武装斗争,从农村包围城市。)”,谋使北京成为社会主义集团的新中心。“大跃进”运动时期提出的“超英赶美”口号,还隐含着“赶超苏联”的目标,只是秘而不宣罢了。

4.资本主义阵营的经济起飞 : 第二次世界大战后到上世纪50年代中期,美国的经济实力独步全球,英、法、日、西德(即德意志联邦共和国)、意等等资本主义国家也迅速恢复了国民经济,相继进入经济起飞的阶段。虽然1953年启动的“第一个五年计划”成效卓著,但与资本主义阵营里的国家、以及苏联的差距仍在拉大。

5.社会主义阵营的赶超热潮 : “1956年,赫鲁晓夫在苏共二十大上代表苏共中央委员会所作的总结报告中提出了和资本主义国家‘和平共处’、‘和平竞赛’等理论,会议要求‘在最短的历史时期内在按人口平均计算的产量方面,赶上并超过最发达的资本主义国家’。④”1956年12月,朝鲜劳动党召开党中央委员会全体会议,倡议、制定并发动了“千里马运动(从1957年开始)”。1957 - 1958年期间,匈牙利、保加利亚、捷克斯洛伐克、德意志民主共和国(即东德)陆陆续续的开启了类似的“赶超计划和运动”。


二.内部因素 : 刚刚建立的中华人民共和国虽不是“一穷二白(中共惯用的话术)”,但也千疮百孔、百业凋零,接收了中华民国政府留下的一副烂摊子。“通货膨胀完全失控;洪水影响了30-40%的耕地;工业产量和食品产量分别骤减到战前最高点的56%和70-75%。⑤”一方面,为了PRC政府的官僚系统的正常运作和社会秩序的基本维持,中共不得不容忍大部分中华民国政府时代的地方官留任原职;另一方面,撤离大陆、死守台湾的中华民国政府重获美国的援助和扶持,且PRC政府“抗美援朝”的(愚蠢的)决策给中华民国政府提供了一个绝处逢生、重振旗鼓的时机,因此,中华民国政府仍有“光复大陆、重振民国”的希望。以上两点直接的威胁着PRC政府的政权。另外,中共内部的派系斗争、知识分子对中共及PRC政府背信弃义和胡作非为的痛恨、民众们对翻来覆去的“运动”和无孔不入的管控忍无可忍而引发的反抗等等,也或多或少的噬食着中共政权的根基。基于此,我们可以得出“大跃进”运动的几点内部因素 : 

1.大生产运动的历史渊源 : 1938年至1947年,中共在其控制区域内发动了一系列军队屯田和鼓励生产的群众运动,以图自救。陕甘宁边区政府专门成立生产委员会,制定和采取有效措施,扩大耕地面积,提高粮食产量,并号召种植经济作物(据《延安日记》和《谢觉哉日记》所撰,鸦片是最重要的一种经济作物,大量种植于陕甘宁边区和晋绥边区)。历经大生产运动,中共从断饷绝粮、饥寒交迫的困境里挣脱出来,实现了基本的自给自足、丰衣足食,这不仅仅让中共及其军事力量熬到颠倒干坤的时机,而且让中共领导人产生了“艰苦奋斗、人定胜天”的狂想。从某种意义上说,大跃进运动是大生产运动的2.0升级版。

2.政治体制的弊病 : 1949年至1953年“新民主主义时期”的中共和PRC政府,遵循着“民主集中制”这一主要原则,“这个词语(指民主集中制)也可引申为表示在决策讨论中畅所欲言,而一旦作出了决定则要不折不扣地坚决服从,无论个人原来的立场如何。⑥”1953年至1956年“社会主义改造时期”通过了一部含四章106条的新宪法,虽然第一章重申了“民主集中制”原则,但第三章的内容里,规定政府有权惩办一切卖国贼和反革命分子,“卖国贼”和“反革命分子”这两顶大帽子让党内、党外的异议者和反对者噤若寒蝉,加之国家主席毛泽东处心积虑的集权于一身,中共和PRC政府已成毛泽东的“一言堂”。1956年至1961年“社会主义建设时期”,“八大二次会议结束后,中共中央成立了财经、政法、外事、科学、文教等小组,毛泽东在有关文件中写道:‘这些小组是党中央的,直隶中央政治局和书记处,向他们直接做报告。大政方针在政治局,具体部署在书记处。只有一个政治设计院,没有两个政治设计院。大政方针和具体部署,都是一元化,党政不分。具体执行和细节决策属政府机构及其党组。对大政方针和具体部署,政府机构及其党组有建议之权,但决定权在党中央。’这一决定实际表明了对‘政’的不信任,所谓‘党政不分’实质是要党直接行‘政’代‘政’。这样,在决策体制上对经济的‘科学计划’管理方式又为随心所欲、非理性的‘命令’‘运动’方式所取代。⑦”也就是说,整个中华人民共和国都成了毛泽东一人的实验室和实验田。

3.毛泽东的愚昧、自负和狂妄 : 一方面毛泽东与赫鲁晓夫互别苗头,另一方面他司马昭上身、袁世凯附体,已至利益熏心、权迷心窍的地步。1956年,毛泽东说“苏联和中国都是社会主义国家,我们是不是可以搞得快点多点,是不是可以用一种更多更快更好更省的办法建设社会主义”,同年赫鲁晓夫抢先一步的提出15年超过美国的口号之后,毛泽东就气急败坏、急不可耐的提出中国要在15年或更短的时间内超过英国,“是不是”的疑问成了“必须”的命令——必须用一种更多更快更好更省的办法建设社会主义。周恩来、刘少奇、陈云、李先念等人皆有“反冒进”的观点,李先念代表国务院向全国人大会议作报告,点明“生产的发展和其他一切事业的发展都必须放在稳妥可靠的基础上。在反对保守主义的时候,必须同时反对急躁冒进的倾向,而这种倾向在过去几个月中,在许多部门和许多地区都已经发生了。急躁冒进的结果并不能帮助社会主义事业的发展,而只能招致损失。”在周恩来和刘少奇的授意之下,《人民日报》先后发表了《读一九五六年的国家预算》和《要反对保守主义,也要反对急躁情绪》两篇社论,这彻底的激怒了毛泽东。毛泽东在《人民日报》的社论上作了“庸俗辩证法”、“尖锐地针对我”、“庸俗马克思主义”等等尖锐异常的批注。1958年1月的南宁会议上,毛泽东一开始就说:“不要提反冒进这个名词——这是政治问题。”“最怕的是六亿人民没有劲,抬不起头来就很不好。”且当众对周恩来说:“你不是反冒进吗?我是反反冒进的。”甚至“把问题提到‘冒进’是马克思主义、‘反冒进’是非马克思主义的高度⑧”。此后,周恩来作了三次羞辱性的检讨,以儆效尤。

4.土地革命和农业集体化 : 1950年6月,PRC政府颁布了《土地改革法》,号召废除“封建剥削的土地所有制”,没收地主和富农的财产和农具并重新分配给无地农民。《土地改革法》落实的过程中,头脑发热的党员干部和报仇心切的农民对地主和富农进行了侮辱、抢夺和残杀,摧毁了传统社会中充当统治精英和主干的士绅阶层。1952年12月土地革命完成之后,1953年的农业集体化运动接踵而至,农业集体化以“互助组”、“初级合作社”和“高级合作社”三个层级逐级递进,至1957年农业集体化运动完成时,全国总计76万到80万合作社,每一合作社含160户或600到700人。“高级合作社”这一种基层农业生产单位,有利于中共和PRC政府的宣传、组织、管控、“教育(或者说“洗脑”)”、镇压和整肃。

5.社会和心理控制 : 中共以“宣传(争取)群众”、“组织群众”和“武装群众”三项指导方针而逐步壮大、坐拥天下的,特别是“宣传(争取)群众”和“组织群众”,早已自成一体和炉火纯青了。中共和PRC政府对农民们威逼利诱、分而治之,对知识分子们身心摧残、监控和洗脑,对公务员们软硬兼施、以致相互监督和相互举报,进而通过群众组织、秘密警察和大众通讯媒体,异常成功的控制并改造了整个社会和民众。


“大跃进”的结果是惨绝人寰、贻害无穷的,堪称人世间最大的(人造)浩劫之一,正如杨继绳于《墓碑 : 中国六十年代大饥荒纪实》一书所言:“这是一场人类历史上空前的悲剧。在气候正常的年景,没有战争,没有瘟疫,却有几千万人死于饥饿,却有大范围的‘人相食’,这是人类历史上绝无仅有的异数。在当时及以后的几十年中,中国的一切书刊、报纸,一切官方文件,都极力回避和掩盖这一大规模的人类悲剧。⑨”以下是“大跃进”所造成的几则触目惊心的恶果 : 

1.大饥荒 : 2005年,曹树基在《中国人口科学》杂志上发表了《1959 - 1961年中国的人口死亡及其成因》一文,他采用了人口学与历史地理学的方法,以1953年、1964年和1982年全国各县市人口普查数据为依据,并参考近千种各省地方志记载的历年人口数,以清代“府”为基本单位进行分析计算,得出了“大跃进”时期的非正常死亡人口数为3250万的结果。此数目与王维志的结论(非正常死亡人数大约3300万 - 3500万人)和杨继绳的结论(非正常死亡人数大约3600万人)几近一致。也就是说,大跃进期间非正常死亡人数在3300万 - 3600万是比较可信的。

2.大破坏 : “从山西晋中的情况来看,据报告,1960年入春以来,‘反革命现行破坏显著增多’,凶杀、投毒、纵火案件较去年同期上升了四倍,破坏的主要目标是公共食堂、幼儿园、饲养场等。⑩”“1960年6月,贵州省江口县农民大量饿死。全县不足10万人却饿死了4万人。在死亡线上挣扎的农民起来暴动。带头人当中有人民公社的党委书记,因为他们家里的孩子也饿死了。手无寸铁的农民很快被现代武装的军队平息。曾开仓放粮的县长饮弹自尽。因虚报粮食产量而激起民变的县委书记也投河身死。1960年5月28日,四川省开县满月公社发生了万全堂暴动。道首刘志善带领300多名道徒抢走公社民兵枪支18条,子弹1200多发,洗劫百货商店、供销社,占领国有林场,打死、打伤党员干部4人,绑架干部16人。捣毁大量救灾物资。1961年7月,反革命集团‘工农民主党’首犯徐世清盗窃‘五四’式手枪5支,子弹50发,在眉山、邛崃、新津等地抢劫粮库和运粮汽车。⑪”

3.文化大革命的跳板 : 1962年的七千人大会上,刘少奇的报告直言“大跃进”不仅没有进,反而倒退了许多,困难形势产生的原因是“三分天灾、七分人祸”且党中央负主要的责任,这让毛泽东怒火中烧、怀恨于心。七千人大会决议向各省传达以后,各级干部提出了很多尖锐的意见,比如“省卫生厅的一个同志说:‘几年来不仅没有成绩,反而把事情搞坏了,不仅物质上搞坏了,政治道德上也搞坏了。’省财贸办公室的同志说:‘这几年发生的问题,不仅地域广,而且时间长,因此,不能说中央的指导思想没有问题。’……省粮食厅解仙清同志说:‘毛主席说过粮食多得吃不了。这没有调查研究。主席可能和斯大林一样,晚年犯了主观主义的错误。’轻工办公室主任杜西林说:‘我听到老百姓骂毛主席,说主席听奸臣的话,不听忠臣的话,还说:毛主席钻到哪个洞里去了,下面发生这么多问题,他知道不知道?’党群系统的干部说:‘甘肃饿死人,毛主席不会不知道。全国不知道死了多少人啊!’⑫”毛泽东看了这些简报上的种种意见之后,感到一股动摇和威胁他个人权力、地位和威望的力量正蠢蠢欲动,素来“与天斗、与地斗、与人斗,其乐无穷”的毛泽东岂会坐以待毙?1962年7月25日召开的北戴河中央工作会议至1962年9月24日召开的八届十中全会期间,毛泽东对刘少奇及其敌对力量展开反击,批判“黑暗风(指刘少奇主持召开的“西楼会议”)”、“单干风(痛批“走资本主义路线”)”和“翻案风(重点是彭德怀)”,且重提阶级斗争,声称“阶级斗争是不可避免的”、“这种阶级斗争,不可避免地要反映到党内来。国外帝国主义和国内资产阶级的影响的存在,是党内产生修正主义思想的社会根源。在对国内外阶级敌人进行斗争的同时,我们必须及时警惕和坚决反对党内种种机会主义的思想倾向”,这些话,都成了文化大革命的指导思想。1966年,天翻地覆、臭名昭著的文化大革命“不可避免”的发生了。

十五年前,孟令伟的一篇旧文《我对当年大跃进运动的看法》让我瞠目结舌,尤以“三狂”之评价入木三分、辛嘲辣讽，其下摘录,以供鉴考和玩味 : 

“现在许多人说起当年的大跃进运动来,认为那真是人类历史上罕见的迷乱和疯狂,当时的那些提法和做法的狂想症状甚至比患有臆想症的精神病患者还要厉害。因为患有臆想症的精神病患者一般局限于想和说,很难真正去做,而‘大跃进’运动则将‘狂想、狂说和狂做’统一了起来。在‘三狂’之下,‘跑步进入共产主义’的试验开始了;亩产粮食超万斤、几十万斤、几百万斤的卫星升天了;全民总动员的钢铁大会战(为实现一年内钢铁产量翻一番)打响了;两年超过英国、10年赶上美国的设想产生了;一个县出一个郭沫若、一年内培养30万个郭兰英的文艺大跃进目标提出来了;‘吃饭不要钱’、几亿农民吃食堂的共产主义式生活过上了;一个县20多天写出30万首诗、6亿人民成为6亿诗人的奇迹发生了。一向以‘彻底的唯物主义者’著称的政治家为此加油鼓劲,科学家为一些‘豪言壮语’寻找科学依据,甚至连世界级的某大科学家也为高产卫星提出了理论根据。一时间全国上下从不同角度、以不同方式、出于不同心态然而都被卷入这场‘狂想、狂说、狂做’的运动中。⑬”
